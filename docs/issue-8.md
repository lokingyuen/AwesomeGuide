# windows10 官網下載、安裝與激活

Hi，大家好。
對於小白用戶，裝系統是比較頭疼的事，所以今天寫一個簡單易懂的裝系統教程。</br>
使用微軟官方提供的工具製作U盤啟動盤，操作簡單，系統純淨，強烈建議小白用戶使用。

[TOC]

## 準備工作

1.至少有 8GB 空間的空白 USB 碟機 (推薦USB3.0 速度更快) </br>
2.Microsoft官網下載原版安裝工具 [Windows 10](https://go.microsoft.com/fwlink/?LinkId=691209)

## 設置啟動項

2.1.因電腦品牌各不相同，操作時最好先Google一下自己的電腦進入UEFI修改啟動選項的按鍵。電腦啟動時，立即重複按下熱鍵，直至出現UEFI啟動選項介面，如未進入，請重新開機重複操作。
> 有些筆記本可能需要加上**Fn**鍵；也有些電腦沒有UEFI啟動，需要進入BIOS修改，具體方法請Google

<img src="https://s2.loli.net/2023/12/11/gM147e5YhAPXI2t.jpg" width="660">
